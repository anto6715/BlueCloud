To run the notebooks, you can either download the OceanRegimesIndicator.zip and run it in your local python environment (using the environment.yml), or you can use the JupyterHub from the BlueCloud VRE.

To do so :

    - Open a JupyterLab instance using the JupyterHub tab in the the VRE
      The notebooks can’t run inside the demonstrator shared folders, so you’ll have to copy the archive inside your Jupyter session :
    - Open a Terminal inside the JupyterLab
    - Copy the archive : cp workspace/VREFolders/MarineEnvironmentalIndicators/notebooks/OceanRegimesIndicator/OceanRegimesIndicator.zip .
    - Change permission : chmod 777 OceanRegimesIndicator.zip
    - Unzip it : unzip OceanRegimesIndicator.zip

You can then play with the notebooks.
